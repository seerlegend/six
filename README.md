# SIX

## 概念

- 爻值对应
  - No　 名称　　符号        铜钱组合
  - 0　　阴爻　　[ - - ]   两个花，一个字
  - 1　　阳爻　　[ --- ]   一个花，两个字
  - 2　　动阴爻　[ -x- ]   三个字
  - 3　　动阳爻　[ -0- ]   三个花

```js
import {Six} from "./six.js";

# 参数 从一爻到六爻(1, 2, 0, 0, 0, 3) 月柱(6,10) 日柱(2,2)
new Six([1, 2, 0, 0, 0, 3], [6, 10], [2, 2]).console()
new Six([3, 0, 0, 0, 0, 0], [0,0], [0, 0]).console()
```
